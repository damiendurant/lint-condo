# lint-condo

A Lint Container for Docker

## Included Linters

Ruby-based:

- scss_lint

Python-based:

- proselint
- yamllint

Node.js-based:

- bootlint
- coffeelint
- csslint
- dependency-lint
- doiuse
- eslint + plugins
- jscs
- jshint
- jslint
- markdownlint
- pug-lint
- remark-lint
- sass-lint
- stylelint
- tern-lint
- write-good
